# 🧋 Boba Bloom

A modern, responsive static beverage website built with **Nuxt** and **Tailwind CSS**.  
The website showcases a colorful collection of drinks, including bubble tea, fruit tea, milk tea, smoothies, frappes, iced beverages, ice cream, and other refreshing desserts.

> **Project Status:** Completed (Static Website)

---

## 📖 About the Project

**Boba Bloom** is a front-end static website designed to provide an attractive and modern user experience for a beverage shop. The project focuses on clean UI, responsive layouts, engaging animations, and visually appealing product presentation.

This project is intended as a portfolio/demo website and does not include backend functionality such as authentication, ordering, or payment processing.

A modified version of this project's frontend implementation was used in a Computer City (CCTL) project.

## 🚀 Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
